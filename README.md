## 👋 Qui suis-je ?

Je suis **Gaël Sassan**, développeur Flutter senior et **Front-End Tech Lead** basé à **Abidjan, Côte d'Ivoire**, avec **6+ ans d'expérience** et **65+ projets livrés** à travers des secteurs aussi variés que la fintech, la santé, l'agriculture, la civic tech et la gestion publique.

Je dirige actuellement les développements front-end chez **Jèko**, et j'opère en parallèle ma propre structure, **WHARPE Corp.**, à travers laquelle je collabore avec des startups, ONG, institutions et entreprises en Afrique et à l'international.

> *"Construire des solutions robustes, maintenables et à fort impact — une app à la fois."*

## 🚀 Projets Notables

### 🔴 Civis-Alerte
Application mobile de signalement citoyen géolocalisé pour le Grand Abidjan. Marqueurs animés, cache intelligent, authentification multi-couche (dont une interface déguisée). Déployé sur TestFlight.

### 🌾 AgriPilote — Investiv Group
Plateforme de réservation de services agricoles par drone. Architecture 3 niveaux : Admin / Client / Télépilote. Refonte complète après retour du board.

### 🏥 Mon Médecin CI
Plateforme de mise en relation patients–médecins–cliniques en Côte d'Ivoire. Documentation complète, pitch guide, intégration Flutter.

### 📊 [Galaxy Stream](https://github.com/ClichyMercury/galaxy-stream)
Application Flutter (iOS & Android) connectée à une API films & séries. Interface soignée, navigation fluide.

### ✅ [MATURUS](https://github.com/ClichyMercury/maturus)
Application de gestion de tâches axée sur la simplicité UX.

### 🤖 [AI Agent — Démarches CI](https://github.com/ClichyMercury/ai-agent-cote-ivoire)
Agent IA TypeScript pour accompagner les citoyens ivoiriens dans leurs démarches administratives. NLP, contexte local, automatisation des parcours.
